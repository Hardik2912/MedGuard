# ADR Reference Sources (FDA/NIH)

This project’s common ADR seed mapping is based on U.S. government/official medical references.

## FDA
- FDA MedWatch (adverse event reporting): https://www.fda.gov/safety/medwatch-fda-safety-information-and-adverse-event-reporting-program
- FDA FAERS (adverse event system): https://www.fda.gov/drugs/fda-adverse-event-reporting-system-faers

## Note on MedDRA
MedDRA is a controlled terminology standard for coding adverse events, not a public “most common ADR by drug” source by itself.
